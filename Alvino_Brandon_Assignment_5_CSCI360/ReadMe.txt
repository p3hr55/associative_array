- The application 'verify_sorts' can be used to make sure that all records are seated correctly
  in the data file. The syntax for this is: "verify_sorts (input file)"

- To run application using memory map use syntax: "sorts (input file) m"
  As long as the first character of the 3rd argument is an 'm' it will execute.

- To run application without memory map then use: "sorts (input file)"
